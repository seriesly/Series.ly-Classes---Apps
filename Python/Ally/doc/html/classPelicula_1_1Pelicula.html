Clase que representa una Pelicula.

NOTAS

Es responsabilidad del usuario usar esta clase como es debido, ya que hay atributos que no siempre se usan y/o inicializan, como por ejemplo el atributo any que se usa en la función tops, pero no en la función user_misPelis, por tanto los objetos Pelicula que devuelva la función user_misPelis no tendrán el campo any que valdrá None y por tanto no se podrá hacer un uso correcto de la función getAny()
